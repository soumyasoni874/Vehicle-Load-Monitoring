
# Vehicle Load Monitoring System 🚚

A complete Data Science + Machine Learning project for monitoring vehicle load efficiency,
detecting overload conditions, and predicting maintenance requirements.

## Features
- Data preprocessing pipeline
- Feature engineering
- Machine learning model training
- Overload detection
- Predictive maintenance analysis
- Streamlit dashboard
- Visualization reports

## Dataset
Dataset used:
`cleaned_logistics_dataset(1).csv`

## Project Structure
```bash
vehicle_load_monitoring_project/
│
├── data/
├── models/
├── notebooks/
├── app.py
├── train_model.py
├── requirements.txt
└── README.md
```

## Installation
```bash
pip install -r requirements.txt
```

## Run Model Training
```bash
python train_model.py
```

## Run Dashboard
```bash
streamlit run app.py
```
