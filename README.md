# Vehicle-Load-Monitoring
A Machine Learning-based Vehicle Load Monitoring System developed using Python, Scikit-learn, and Streamlit. This project predicts whether a vehicle is operating safely or overloaded based on parameters such as load capacity, actual load, engine temperature, fuel consumption, vibration levels, failure history, and predictive risk score.<img width="1920" height="1020" alt="vehicle_load _monitoring_img" src="https://github.com/user-attachments/assets/ee915cb6-9c27-453b-942b-a03e4b613ceb" />
